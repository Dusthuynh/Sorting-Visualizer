Thông tin giải thuật

Code tương ứng (A = array)


  n = length(A)
  for i = 0 to (n - 1) do
    min_index = i
    for j = i + 1 to (n - 1) do
      if compare(A[min_index], A[j])
        min_index = j
    end for j
    swap (min_index, i)
  end for i
